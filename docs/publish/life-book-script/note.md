# 你的人生剧本是哪本古书？

我觉得“测你像谁”已经有点挤了。

但“测你的人生剧本”还有空间。

因为人不一定想知道自己像谁。
人更想知道：

我这一生，到底像哪种故事？

有的人是《史记》型。
人生必须有转折、有重量、有值得写下来的名字。

有的人是《资治通鉴》型。
不冲动，不迷信运气，所有选择都要先看规律和代价。

有的人是《三国志》型。
天生对机会、位置、竞争和联盟敏感，不太甘心一直在局外。

有的人是《世说新语》型。
不一定追求最大胜利，但很在意风格、审美、风骨和一句话的锋芒。

有的人是《红楼梦》型。
细腻、深情，能读懂关系里的盛衰，也容易被人心牵动。

还有人是《山海经》型。
奇异、非标、想象力很强，不适合被普通剧本解释。

## 个人售卖详情

你有没有一种感觉：

有些人的人生像人物传记。
有些人的人生像成败复盘。
有些人的人生像乱世入局。
有些人的人生像情感长卷。

这份《32题古籍人生剧本测试》，会用 6 本经典古书作为人格原型，帮你看清自己的人生底稿更接近哪一种叙事。

完成测试后，你会获得一份个人报告，内容包括：

6 类结果包括：

《史记》型：人物立传，有重量、有转折

《三国志》型：机会入局，重位置和竞争
