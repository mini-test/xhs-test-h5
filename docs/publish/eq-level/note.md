# 情商水平摸底考试

有些人不是没情商。
是懒得把情商用在每一个场面上。

有些人看起来特别会做人。
其实不是圆滑，是太会读空气。

所以这次我做了一个《32 题情商水平摸底考试》。

它不问你“会不会说漂亮话”。
它看的是你在真实社交场景里的第一反应：

饭桌冷场时，你会不会救场？
别人手机外放尴尬声音时，你会不会提醒？
群里有人发错消息时，你会装没看见，还是私下提醒？
朋友被冷落时，你会主动递台阶，还是觉得每个人都该自己找位置？

完整测试会从 5 个维度生成报告：

情绪感知力
氛围阅读力
人心共情力
向下兼容力
冲动控制力

最后会得到一个你的情商类型：

气氛雷达型
体面周旋型
万般皆乐型
低耗直球型

我觉得这个测试最有意思的地方是：
它不会简单说你“高情商”或“低情商”。

因为很多人的真实情况是：

你明明很会读空气，但已经不想再救场了。
你不是不会说话，只是不想每次都照顾别人。
你看起来很轻松，其实是在用幽默化解压力。
你看起来很直接，其实是在保护自己的社交电量。
