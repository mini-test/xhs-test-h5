# 你是真的想辞职，还是只想被好好对待？

先别急着提离职。

有时候你说「我想辞职」，不是真的马上想走。

是你太久没有被认真听过。

做了很多事，却像理所当然。
出了问题才找你，平时看不见你。
边界一次次往后退，最后所有临时需求都变成你来扛。

但也有另一种情况：
你不是委屈一阵子。
你是真的已经不想把时间继续交给这里了。

所以这组题不是问你「该不该辞职」。

它想帮你分清：

你是真的想离开，
还是只是想被好好对待一次。

完整测试共 32 道，图里先放前 5 道。

做完会看到你更接近哪一种状态：

真想离开型
渴望被善待型
耗竭降噪型
边界谈判型

也可以把你选的编号留在评论区，我后面按高频组合继续拆。
