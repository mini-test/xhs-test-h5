# 你的反杀能力，放在古代是什么段位？

真正高级的反杀，不是当场吵赢。
也不是把谁踩下去。

而是你被误解、被甩锅、被抢功、被阴阳的时候，
还能稳住情绪，保住证据，看清谁在试探你，
最后把局面重新拿回来。
