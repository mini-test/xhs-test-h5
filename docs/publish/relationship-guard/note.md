# 你在人际关系里，最该防谁？

有些人不是一上来就伤害你。

他们更常见的方式是：
让你内疚，让你帮忙，让你站队，让你不好意思拒绝。

到最后你才发现，自己不是不会社交，而是边界太容易被拿捏。

我做了一个《32题人际边界测试》。

它不是教你怀疑所有人，也不是让你给身边人贴标签。

它只是帮你看清：
你在人际关系里，最容易被哪一类人消耗。

测试会从 8 个维度看你的真实反应：

初始信任
边界测试
情绪反应
利益交换
信息与圈层
冲突处理
亲密关系
自我盲区

最后会生成 4 类结果之一：

情绪债主型风险：最该防总让你内疚的人
利益索取型风险：最该防只在需要你时出现的人
信息操控型风险：最该防用秘密和站队带你入局的人
边界侵占型风险：最该防借着熟悉一点点越界的人

报告里会写清楚：

你为什么容易中招
你最明显的边界弱点
哪类关系信号需要警惕
适合你的拒绝方式
未来 90 天边界练习建议

## 个人售卖详情

你是不是也经常在关系里遇到这种情况：

明明不想答应，但对方一失望你就心软。
明明已经帮了很多，但对方下次还是继续找你。
明明只是听别人吐槽，最后却被卷进是非和站队。
明明只是熟人，对方却越来越自然地占用你的时间、隐私和情绪。

这份《32题人际边界测试》，会从真实关系场景里帮你看清：
你最容易被哪类人消耗，以及你该从哪里开始建立边界。

完成测试后，你会获得一份个人结果报告，内容包括：

一句话主结论
你的人际风险类型
为什么你容易被这类人拿捏
你的关键边界弱点
高风险关系信号提醒
不建议继续忍让的场景
未来 90 天边界练习建议
