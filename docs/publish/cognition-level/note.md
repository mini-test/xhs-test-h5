# 测测你的认知在第几层？

同一件事，有人只看到热闹。

有人能看到经验。

有人能看到系统。

还有人能看见：自己为什么会这样想。

这份《32 题认知层级测试》不是测你聪不聪明，也不是给人分高低。

它更想帮你看清：

你面对信息、争论、选择和复杂问题时，第一反应更接近哪一种。

是先被热度、评论和表达强度带着走；

还是用自己的经验快速判断；

还是会拆目标、变量、约束和反例；

又或者，你会先观察自己为什么会认同、反感、焦虑或防御。

完整测试一共 32 题，会从 8 个模块看你的认知方式：

信息进入、判断决策、复杂问题、情绪偏见、学习升级、人际观点、现实选择、反思迭代。

做完后会生成一份个人报告，包括：

一句话主结论

你的认知层级类型

主要思维盲区

适合你的升级路径

以及下一步可以练的具体动作。

可以先凭第一反应做一遍，再去完整链接看报告。

本测试为趣味自我观察工具，非科学测评、心理诊断或能力评级，仅供娱乐和复盘参考。
