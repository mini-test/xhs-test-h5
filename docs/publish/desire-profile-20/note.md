# 测测你的欲望最强是哪一种？

你以为你在做测试，
其实你的欲望早就藏不住了。

很多人不敢承认自己有欲望。

想被爱，怕显得太需要。
想赚钱，怕被说太现实。
想吃好睡好，怕被说不自律。
想确认关系，怕被说控制欲强。

但欲望不是问题。
它只是你内心最缺、最怕失去、最想被满足的地方。

这份《20 题欲望档案测试》会从 4 个方向帮你看清：

爱欲：你有多需要亲密、偏爱和被选择感。

物欲：你有多需要资源、体面和现实安全感。

食欲：你有多需要感官满足、身体补给和即时安慰。

占有欲：你有多需要确定关系、清楚边界和掌控感。

测完会生成一份专属欲望档案，包括：

一句话主结论
你的欲望主轴
副欲望倾向
最容易被触发的场景
欲望背后的真实需求
当前内耗提醒
最近 30 天调整建议

适合最近总觉得自己很贪心、很敏感、很容易上头，或者明明什么都不缺但还是不满足的人。
