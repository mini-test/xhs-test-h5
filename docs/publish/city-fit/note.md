# 测一测你适合的城市｜你需要什么样的生活？

你以为自己在选城市，
其实是在选一种每天都要重复的生活。

同样是换城市：

有人在高密度机会里越忙越有劲；
有人只想要准点下班、好好吃饭和有烟火气的关系；
有人把专业圈和文化内容当成长燃料；
也有人只有在阳光、绿地和低压节奏里，才真正恢复状态。

所以，适合你的城市，不一定是别人都在推荐的那一座。

这份测试不告诉你哪个城市“最好”，而是先帮你分辨：
你现在更需要机会、创作、专业、松弛、秩序，还是恢复。

完整测试共 32 题，会从 8 个生活维度观察你的倾向：

生活节奏、职业发展、气候与身体、生活成本、关系归属、文化审美、迁居决策、日常适配。

你可能会测出：

A｜深圳型 · 机会加速器
适合把行业密度、行动速度和职业跃迁放在前面的人。

B｜杭州型 · 创意创业场
适合一边生活，一边做内容、项目或小生意的人。

C｜上海型 · 专业深耕者
适合需要专业资源、文化内容和长期成长空间的人。

D｜成都型 · 松弛生活派
适合希望工作之外，仍然保留关系、吃饭和日常快乐的人。

E｜苏州/宁波型 · 秩序宜居派
适合重视稳定、可控成本、家庭规划和长期积累的人。

F｜昆明/厦门型 · 自然疗愈派
适合把阳光、空气、睡眠和身体恢复视为生产力的人。

## 个人售卖详情

这不是简单告诉你“适合去哪个城市”的标签测试。

1. 你的主要城市生活类型和次要倾向。
2. 你真正想从城市里获得什么。
3. 适合你的城市生活场景和示例方向。
4. 迁居时最容易忽略的风险与现实矛盾。
5. 如果继续留在原环境，可能出现的状态变化。
6. 一份可以执行的 90 天试住、预算和验证计划。
