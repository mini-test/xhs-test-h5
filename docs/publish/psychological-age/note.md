# 你的心理年龄多少岁？

有时候真的会怀疑：

明明身份证上已经是大人了，
可心里还是会因为一句话想很久。

也可能相反。
年纪不算大，却总是在关系里负责稳定情绪，
在工作里负责收拾残局，
在家人朋友面前习惯先把自己放到后面。

所以这次做了一个《32题心理年龄测试》。

它不是简单问你“成熟不成熟”，
而是从 8 个维度看你现在更常使用哪种心理模式：

好奇心 / 情绪稳定 / 社交从容 / 责任担当
抗逆力 / 生活热情 / 包容度 / 自我接纳

做完后会生成一份个人结果：

心理年龄 vs 实际年龄
8维心理画像
你最强的心理能力
你最容易卡住的地方
你的社交原型和同频拍档
未来一段时间的成长建议

前 5 题我放在图里了。
建议不要选“我应该是什么样的人”，
就按第一反应选真实的自己。
