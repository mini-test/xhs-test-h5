# 测一下，你是什么灵根？

修仙界有句话：灵根定终身。

但我觉得，更准的不是你“看起来像哪种人”，而是你在压力里的第一反应。

遇到强过你的人，你是先观察弱点，还是直接硬冲？

学一样东西，你是死磕到底，还是哪里有捷径就走哪里？

在团队里，你更像输出、军师、奶妈、先锋，还是肉盾？

很多人不是没天赋，是不知道自己的天赋属于哪一种。

火灵根的人，爆发力强，被逼急了反而最容易翻盘。
