# 测一下，你的隐藏天赋是哪一种？

有些人从小就很会说话。

有些人看到复杂问题，脑子会自动拆结构。

有些人走进一个空间，立刻知道哪里不舒服。

还有些人不一定外向，但特别会读气氛、看情绪、感知别人没说出口的东西。

我越来越觉得，很多人不是没有天赋。

是一直把自己的天赋当成「普通反应」。

这份《24 题隐藏天赋测试》，不是问你喜欢什么学科，也不是让你硬选一个标签。

它会从生活场景里看你：

你遇到混乱先看结构，还是先稳住人？

你被压力卡住时，是写下来、动起来，还是需要独处？

你最看不下去的是表达混乱、逻辑不清，还是审美粗糙？

测完会生成一份个人天赋报告：

你的主天赋类型

你的副倾向

你为什么容易被低估

你适合在哪些场景发力

以及 30 天的小行动建议

如果你最近总觉得自己好像有点能力，但不知道该往哪里用，可以测一下。

别想太久，凭第一反应选。

测试仅供娱乐和自我探索，不做科学诊断。
