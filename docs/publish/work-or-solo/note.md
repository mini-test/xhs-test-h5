# 你适合上班还是单干？

你有没有想过：

自己到底是不适合上班，
还是只是不适合现在这份工作？

很多人想单干，不是因为已经准备好创业了，
而是因为现在太累、太憋屈、太想逃离。

但单干不是“不上班”。

它会把压力从领导、同事和公司，
换成客户、现金流、获客、交付、售后和自我管理。

这份《32 题上班/单干适配测试》，会从 8 个维度帮你拆清：

你是真的适合独立经营，
还是更适合换一个正常的组织环境；

你是已经具备单干底盘，
还是只需要先做副业验证；

你现在想离开，
到底是因为看见了新路径，
还是因为被当前工作消耗到想逃。

测试完成后，你会得到一份个人报告，包括：

一句话主结论  
你的职业路径类型  
适合上班/单干的核心原因  
当前最大风险提醒  
关键指标分析  
未来 90 天行动建议

结果类型有 4 种：

平台借力上班型  
独立经营单干型  
副业验证过渡型  
自由幻想慎重型

适合正在纠结这些问题的人：

要不要裸辞  
要不要做自由职业  
要不要把副业做大  
是不是不适合上班  
到底该换工作，还是该单干
